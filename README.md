# SKY Web Studio

Luxury website concept scaffold built with Next.js 15 + TypeScript + Tailwind CSS.

## Included
- Premium dark-mode homepage scaffold
- Vietnamese high-end copywriting for homepage
- Service / portfolio / pricing / testimonial content source
- Luxury visual system and reusable section structure

## Next recommended steps
1. Install dependencies: `npm install`
2. Run local dev: `npm run dev`
3. Add shadcn/ui, GSAP, Lenis, and Framer Motion refinements
4. Build internal pages: services, portfolio, pricing, about, contact
5. Connect CMS: Sanity or Payload
6. Add quote configurator, booking, SEO schema, and analytics
