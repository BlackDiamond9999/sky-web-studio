# IMPLEMENTATION GUIDE - SKY Web Studio

## 1. Phase 1 - Foundation
- Chốt logo wordmark + monogram
- Chốt typography: Inter + Satoshi/Manrope
- Chốt premium image direction and mockup style
- Chốt sitemap và content model

## 2. Phase 2 - UX/UI
- Dựng wireframe từng trang
- Thiết kế high-fidelity trong Figma
- Define motion system: reveal, parallax, card hover, premium transitions

## 3. Phase 3 - Frontend
- Setup Next.js App Router
- Xây section reusable
- Thêm Framer Motion / Lenis / GSAP
- Tối ưu responsive mobile-first

## 4. Phase 4 - CMS + Forms
- Sanity schemas: services, projects, testimonials, blog, settings
- Multi-step consultation form
- Quote configurator
- Booking integration: Calendly or custom route

## 5. Phase 5 - SEO + Launch
- Metadata per page
- JSON-LD schema
- OG images
- sitemap.xml + robots.txt
- Vercel deployment
- Google Search Console + Analytics

## Deliverables in current scaffold
- Homepage structure
- Luxury color system
- Premium Vietnamese copywriting
- Reusable content file
- Project-ready folder structure basis
